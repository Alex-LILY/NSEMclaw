│                              NSEM 记忆进化管线 (Evolution Pipeline)                           │
└─────────────────────────────────────────────────────────────────────────────────────────────┘

                                    ┌─────────────────────────────────────────┐
                                    │  PeriodicMaintenanceService.run()      │
                                    │  定时任务: 每小时/每日/每周             │
                                    └──────────────────┬────────────────────────┘
                                                       │
                                    ┌──────────────────┴────────────────────────┐
                                    ▼                                             ▼
                         ┌─────────────────────┐                 ┌─────────────────────┐
                         │ 调度进化任务         │                 │                      │
                         └─────────────────────┘                 └─────────────────────┘
                                                       │
                                                       ▼
┌───────────────────────────────────────────────────────────────────────────────────────────┐
│                           任务1: 记忆衰减 (decay) - 每小时                                   │
├───────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                            │
│  ┌────────────────────────────────────────────────────────────────────────────────────┐   │
│  │ EvolutionEngine.decay()                                                          │   │
│  │                                                                                    │   │
│  │ 目标: 模拟人类记忆自然衰退，强化重要记忆，淡化不重要记忆                           │   │
│  │                                                                                    │   │
│  │ 公式:                                                                             │   │
│  │                                                                                    │   │
│  │   newStrength = oldStrength * (1 - decayRate * timeDiff)                       │   │
│  │                                                                                    │   │
│  │   decayRate = 0.01 / 小时 (可配置)                                               │   │
│  │                                                                                    │   │
│  │ 示例:                                                                             │   │
│  │   10小时未访问的 Atom:                                                           │   │
│  │   strength = 1.0 * (1 - 0.01 * 10) = 0.9                                        │   │
│  │                                                                                    │   │
│  │ 特殊情况:                                                                         │   │
│  │   - 重要记忆 (importance > 0.8): decayRate *= 0.1  // 衰减慢                   │   │
│  │   - 频繁访问 (accessCount > 10): decayRate *= 0.5                               │   │
│  │                                                                                    │   │
│  └────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                            │
└───────────────────────────────────────────────────────────────────────────────────────────┘
                                           │
                                           ▼
┌───────────────────────────────────────────────────────────────────────────────────────────┐
│                           任务2: 记忆修剪 (prune) - 每日                                    │
├───────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                            │
│  ┌────────────────────────────────────────────────────────────────────────────────────┐   │
│  │ EvolutionEngine.prune()                                                          │   │
│  │                                                                                    │   │
│  │ 目标: 删除低价值记忆，释放存储空间                                                │   │
│  │                                                                                    │   │
│  │ 修剪条件 (满足任一):                                                              │   │
│  │                                                                                    │   │
│  │ 1. strength < 0.1  // 强度过低                                                    │   │
│  │ 2. importance < 0.2 且 lastAccessed > 30天                                        │   │
│  │ 3. accessCount = 1 且 created > 7天                                             │   │
│  │                                                                                    │   │
│  │ 流程:                                                                             │   │
│  │                                                                                    │   │
│  │   1. 扫描所有低于阈值的 Atom                                                     │   │
│  │   2. 软删除 (标记为 deleted)                                                    │   │
│  │   3. 30天后真正删除 (可恢复)                                                     │   │
│  │                                                                                    │   │
│  └────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                            │
└───────────────────────────────────────────────────────────────────────────────────────────┘
                                           │
                                           ▼
┌───────────────────────────────────────────────────────────────────────────────────────────┐
│                           任务3: 记忆场合并 (merge-fields) - 每日                           │
├───────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                            │
│  ┌────────────────────────────────────────────────────────────────────────────────────┐   │
│  │ EvolutionEngine.mergeFields()                                                     │   │
│  │                                                                                    │   │
│  │ 目标: 合并相似的记忆场，减少冗余                                                  │   │
│  │                                                                                    │   │
│  │ 条件:                                                                             │   │
│  │   两个场 F1, F2 满足:                                                             │   │
│  │   - 距离: dist(F1.centroid, F2.centroid) < threshold                            │   │
│  │   - 合并后大小 < maxFieldSize                                                     │   │
│  │                                                                                    │   │
│  │ 流程:                                                                             │   │
│  │                                                                                    │   │
│  │   1. 计算新质心: newCentroid = weightedAvg(F1, F2)                              │   │
│  │   2. 合并原子: atoms = F1.atoms + F2.atoms                                      │   │
│  │   3. 更新场属性: radius, vitality                                                │   │
│  │   4. 删除旧场                                                                     │   │
│  │                                                                                    │   │
│  └────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                            │
└───────────────────────────────────────────────────────────────────────────────────────────┘
                                           │
                                           ▼
┌───────────────────────────────────────────────────────────────────────────────────────────┐
│                           任务4: 存储优化 (optimize-storage) - 每周                         │
├───────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                            │
│  ┌────────────────────────────────────────────────────────────────────────────────────┐   │
│  │ EvolutionEngine.optimizeStorage()                                                 │   │
│  │                                                                                    │   │
│  │ 目标: 优化向量存储，提高查询性能                                                  │   │
│  │                                                                                    │   │
│  │ 优化项:                                                                           │   │
│  │                                                                                    │   │
│  │ 1. 压缩: Float16 → Int8 量化 (节省 50% 空间)                                    │   │
│  │ 2. 索引优化: 重建 HNSW 索引                                                      │   │
│  │ 3. 碎片整理: 合并小文件                                                          │   │
│  │ 4. 冷热分离: 将低频访问数据移到慢速存储                                          │   │
│  │                                                                                    │   │
│  └────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                            │
└───────────────────────────────────────────────────────────────────────────────────────────┘
                                           │
                                           ▼
┌───────────────────────────────────────────────────────────────────────────────────────────┐
│                           任务5: 索引重建 (rebuild-index) - 每月                             │
├───────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                            │
│  ┌────────────────────────────────────────────────────────────────────────────────────┐   │
│  │ EvolutionEngine.rebuildIndex()                                                    │   │
│  │                                                                                    │   │
│  │ 目标: 重建向量索引，保证搜索质量                                                  │   │
│  │                                                                                    │   │
│  │ 流程:                                                                             │   │
│  │                                                                                    │   │
│  │   1. 导出所有有效 Atom                                                            │   │
│  │   2. 删除旧索引                                                                   │   │
│  │   3. 重新构建 HNSW 索引                                                           │   │

│  │                                                                                    │   │
