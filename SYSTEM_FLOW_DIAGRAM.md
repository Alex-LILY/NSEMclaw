## ⚙️ 配置参数

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           系统配置 (nsemclaw.json)                           │
└─────────────────────────────────────────────────────────────────────────────┘

agents:
  defaults:
    nsem:
      enabled: true                    # 启用 NSEM 认知核心
      resourceMode: "performance"      # 资源模式: minimal/balanced/performance
      models: ["embedding"]             # 启用的模型
      autoDownloadModels: true         # 自动下载模型
      ingestConversations: true        # 自动摄入对话

  main:                                # Agent "main" 配置
    nsem:
      mode: "fusion"                   # 存储模式

memorySearch:
  enabled: true                        # 启用记忆搜索
  provider: "local"                    # 嵌入提供者
  model: "hf:...embeddinggemma-300m"   # 模型路径
  
  local:
    modelCacheDir: "~/.nsemclaw/models"  # 模型缓存目录
  
  sync:
    onSessionStart: true               # 会话开始时同步
    onSearch: true                     # 搜索时同步
    watch: true                        # 监视文件变化
    watchDebounceMs: 5000              # 防抖时间
```

## 🔧 单例保障机制

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      getNSEMFusionCore 单例保障                              │
└─────────────────────────────────────────────────────────────────────────────┘

并发调用场景:
  调用1: getNSEMFusionCore("main", config) 
  调用2: getNSEMFusionCore("main")          ← 几乎同时

处理流程:
  ├─ 检查 fusionCoreInstances.has("main")? 
  │   └─ 否 (首次调用)
  │
  ├─ 检查 fusionCoreLocks.has("main")?
  │   └─ 否 (无锁)
  │
  ├─ 创建 Promise (锁定)
  │   └─ fusionCoreLocks.set("main", createPromise)
  │
  ├─ 执行创建
  │   ├─ createNSEMFusionCore()
  │   ├─ core.initialize()
  │   └─ fusionCoreInstances.set("main", core)
  │
  └─ 释放锁
      └─ fusionCoreLocks.delete("main")

调用2 处理:
  ├─ 检查 fusionCoreInstances.has("main")?
  │   └─ 否 (创建中)
  │
  ├─ 检查 fusionCoreLocks.has("main")?
  │   └─ 是! 返回现有 Promise
  │
  └─ 等待创建完成，返回同一实例

结果: 两次调用返回同一个 NSEMFusionCore 实例 ✅
```
