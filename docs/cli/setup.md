# `nsemclaw setup`

Initialize `~/.nsemclaw/nsemclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
nsemclaw setup
nsemclaw setup --workspace ~/.nsemclaw/workspace
```

To run the wizard via setup:

```bash
nsemclaw setup --wizard
```
